4.4复制两个地址
dmg url : https://github.com/jiangfei84/ElitechLog-Update/releases/download/2.0/ElitechLog.dmg

xml url: https://github.com/jiangfei84/ElitechLog-Update/releases/download/2.0/ElitechLog.xml

这两个地址是固定死的，按照上面的操作来
dmg url是新版本的下载地址
xml url是程序info.plist提前配置好的地址
以后有新的版本3.0，4.0也最好点击编辑2.0这个里面，a按照4.3，删除老版本，上传新版本即可

4.4 xml说明

dmg url要填入xml中

其中 channel 中

title 是工程名字；
link 是本xml 链接地址；
description 是跟新描述，将出现在更新面板中；
pubDate 是发布日期 ；
sparkle:dsaSignature 后面跟的是第五步生成的数字签名；
enclosure url 后面是 安装包地址；
sparkle:version 后面是安装包的版本；
length=“436264” 指更新包的大小，上传上去前用 终端 ls -al 命令可查看；
type=“application/octet-stream” 是下载类型，使用afn 的同学可能会了解，一般不用修改。
